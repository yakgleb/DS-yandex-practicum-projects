# Yandex.Practicum Data Science проекты
Репозиторий моих проектов, выполненных в рамках обучения на курсе "Специалист по Data Science" от "Яндекс Практикум".


| Проект                                                                                                                                          | Описание                                                                        | Навыки и инструменты                                                                                                                                |
